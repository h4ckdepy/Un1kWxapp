## 介绍

自动化的微信小程序源码获取平台,速来公众号《明晚吃火锅》体验使用

作者 depy/NLEG

## 使用方法

1.关注公众号

2.输入小程序全名@wxapp 

3.等待任务结束 获取下载链接
